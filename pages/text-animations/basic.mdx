## Dépendances

Pour utiliser les animations de texte, vous aurez besoin de quatres dépendances JS et une dépedance CSS :

- [GSAP](https://www.jsdelivr.com/package/npm/gsap)
- [Splitting](https://www.jsdelivr.com/package/npm/splitting)
- [GSAP Custom Easing](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/CustomEase.min.js)
- [GSAP ScrollTrigger](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js)
- [MM-TextReveal.css](https://github.com/Rakido/mm-animation-library/blob/main/css/data-text-reveal.css)

## Animation de texte

Pour animer un texte, vous aurez besoin de deux attributs principaux : `data-scroll-text-reveal` et `data-splitting`.

```html
<h1 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="fade-in"
>
  Ici l'animation est appliquée à chaque caractère
</h1>
```

L'attribut `data-scroll-text-reveal` initialise le système d'animation, tandis que `data-splitting="chars"` indique à la librairie de diviser le texte en caractères individuels.

<NxTextAnimation 
  data-scroll-text-reveal 
  data-animate="fade-in"
  data-splitting="chars"
>
  Ici l'animation est appliquée à chaque caractère
</NxTextAnimation>
