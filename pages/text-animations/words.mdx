# Words animations

import NxTextAnimation from '../../components/NxTextAnimation'

## Animation mot à mot

Pour animer mot à mot il faut utiliser `data-splitting="words"`.

```html
<h1 
  data-scroll-text-reveal 
  data-splitting="words"
  data-animate="slide"
>
  Animer mot par mot
</h1>
```

<NxTextAnimation 
  data-scroll-text-reveal 
  data-splitting="words"
  data-animate="slide"
>
  Animer mot par mot
</NxTextAnimation>
