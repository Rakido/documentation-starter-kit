## Animation de base

La bibliothèque permet de créer des animations de marquees infinies.

Pour créer une animation parallax, vous aurez besoin de :
- `data-parallax` sur le conteneur
- `data-animate="marquee"` sur le conteneur

Le parent doit être en `width: max-content` et contenu dans un wrapper.

```html
<div class="marquee-wrapper" 
     data-parallax 
     data-animate="marquee" 
">
  <h1 style="margin:0; font-size:140px;">Hello World</h1>
  <h1 style="margin:0; font-size:140px;">Hello World</h1>
  <h1 style="margin:0; font-size:140px;">Hello World</h1>
  <h1 style="margin:0; font-size:140px;">Hello World</h1>
  <h1 style="margin:0; font-size:140px;">Hello World</h1>
</div>
```
