# Effet Grid

import NxImageReveal from '../../components/NxImageReveal'

## Configuration de base

L'effet grid divise l'image en cellules qui s'animent individuellement.
Définissez le nombre de colonnes avec `data-grid-columns` et le nombre total de cellules avec `data-grid-cells`.

```html
<div 
  data-scroll-image-reveal 
  data-animate="grid"
  data-grid-cells="9"
  data-grid-columns="3"
>
  <img data-scroll-image-reveal-target src="image.jpg" />
</div>
```

  data-grid-cells="9"
  data-grid-columns="3"

/>


## Apparence

### Couleur des cellules
Personnalisez la couleur avec `data-grid-color` :

```html
<div 
  data-scroll-image-reveal 
  data-animate="grid"
  data-grid-color="#ff4444"
>
  <img data-scroll-image-reveal-target src="image.jpg" />
</div>
```

## Animation

### Direction du Stagger
Contrôlez la séquence d'animation avec `data-grid-stagger-direction` :

```html
<div 
  data-scroll-image-reveal 
  data-animate="grid"
  data-grid-stagger-direction="random"
>
  <img data-scroll-image-reveal-target src="image.jpg" />
</div>
```

Options disponibles :
- `start` : Depuis le début
- `center` : Depuis le centre
- `end` : Depuis la fin
- `random` : Aléatoire

<NxImageReveal
  data-animate="grid"
  data-grid-stagger-direction="random"
  src="/images/demo-1.jpg"
/>

### Easing personnalisé
Ajustez l'easing avec `data-grid-easing` :

```html
<div 
  data-scroll-image-reveal 
  data-animate="grid"
  data-grid-easing="power3.inOut"
>
  <img data-scroll-image-reveal-target src="image.jpg" />
</div>
```
