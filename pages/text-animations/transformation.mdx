import NxTextAnimation from '../../components/NxTextAnimation'

## Rotation

L'attribut `data-rotate` donne une rotation de départ aux éléments :

```html
<div 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-stagger="0.1"
  data-rotate="40"
>
  Les caractères s'animent avec 40° de rotation au départ
</div>
```

<NxTextAnimation 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="fade-in"
  data-stagger="0.1"
  data-rotate="40"
>
  Chaque lettre apparaît une par une
</NxTextAnimation>

## Skew

L'attribut `data-skew` donne une inclinaison de départ aux éléments :

```html
<div 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-stagger="0.1"
  data-skew="40"
>
  Les caractères s'animent avec 40° d'inclinaison au départ
</div>
```
