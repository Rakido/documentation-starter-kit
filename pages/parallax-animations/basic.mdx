## Animation de base

Le parallax permet d'animer des images avec un effet de profondeur au scroll.

Pour créer une animation parallax, vous aurez besoin de :
- `data-parallax` sur le conteneur
- `data-image-parallax` sur le conteneur

```html
<div class="parallax-container"
     data-parallax 
     data-image-parallax >
  <img 
    src="image.jpg" 
  />
</div>
```

## Nécessite du CSS

```css
.parallax-container {
  position: relative;
  overflow: hidden;
}
```
