# Transformations

## Rotation

Ajoutez une rotation avec `data-rotate` :

```html
<ul 
  data-stagger-reveal 
  data-stagger="0.1"
  data-rotate="45"
>
  <li>Accueil</li>
  <li>Services</li>
  <li>Portfolio</li>
  <li>Contact</li>
</ul>
```

<NxStaggerAnimation 
  data-stagger-reveal 
  data-stagger="0.1"
  data-rotate="45"
>
  {[1, 2, 3, 4].map((item) => (
    <div 
      key={item}
      data-stagger-item
      className="nx-bg-gray-700 nx-p-4 nx-rounded-lg nx-mb-4"
    >
      Item {item}
    </div>
  ))}
</NxStaggerAnimation>

## Scale

Utilisez `data-scale` pour une animation d'échelle, la valeur spécifiée est la valeur de début de l'animation :

```html
<ul 
  data-stagger-reveal 
  data-animate="slide"
  data-direction="-y"
  data-scale="0.5"
>
  <li></li>
  <li></li>
  <li></li>
</ul>
```

<NxStaggerCardsAnimation 
  data-animate="slide"
  data-direction="-y"
  data-scale="0.5"
/>

## Paramètres

Pour limiter la transformation à un axe vous pouvez utiliser `data-scale-axis`.
Pour contrôler le point de transform-origin vous pouvez utiliser `data-scale-position`.

```html
<div data-stagger-reveal data-scale="0.5" data-scale-axis="x" data-scale-position="right">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

<NxStaggerCardsAnimation 
  data-scale="0.5"
  data-scale-axis="x"
  data-scale-position="right"
/>
