# Easing Personnalisé

import NxTextAnimation from '../../components/NxTextAnimation'

## Utilisation des Easings GSAP

L'easing de l'animation peut être personnalisé avec `data-easing` :

Les courbes d'easing GSAP les plus courantes sont :

- `power1.out` : Accélération douce
- `power2.inOut` : Accélération et décélération modérées
- `power3.in` : Démarrage lent puis accélération forte
- `back.out` : Effet de rebond léger à la fin
- `elastic.out` : Effet élastique oscillant

Pour plus de détails sur les easings disponibles, consultez la [documentation GSAP](https://gsap.com/docs/v3/Eases/).


Voici un exemple avec elastic.out :


```html
<h3 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-easing="elastic.out"
>
  Animation en easing elastic.out
</h3>
```

<NxTextAnimation 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-easing="elastic.out"
>
  Animation en easing elastic.out
</NxTextAnimation>

## Cubic-Bezier Personnalisé

Pour un contrôle précis, utilisez des valeurs cubic-bezier personnalisées :

```html
<p 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="fade-in"
  data-easing="0.7,0,0.3,1"
>
  Animation avec courbe d'easing personnalisée
</p>
```

<NxTextAnimation 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-easing="0.7,0,0.3,1"
>
  Animation avec courbe d'easing personnalisée
</NxTextAnimation>

## Easing SVG

Il est possible d'utiliser des courbes SVG en easing :

```html
<h3 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-easing="M0,0 C0.769,-0.265 -0.021,0.933 0.24,1.081 0.56,1.261 0.696,1.001 1,1"
>
  Animation avec un easing SVG
</h3>
```
