import NxTextAnimation from '../../components/NxTextAnimation'

## Effet de Décalage

L'attribut `data-stagger` contrôle le délai entre l'animation de chaque caractère :

```html
<div 
  data-scroll-text-reveal 
  data-animate="slide"
  data-splitting="chars"
  data-stagger="0.5"
>
  Les caractères s'animent lentement
</div>
```

  data-splitting="chars"

</NxTextAnimation>

## Durée et Délai

Contrôlez le timing global avec `data-duration` et `data-delay` :

```html
<span 
  data-scroll-text-reveal 
  data-splitting="chars"
  data-animate="slide"
  data-duration="5"
  data-delay="1"
>
  Animation très lente qui commence après 1s
</span>
```
