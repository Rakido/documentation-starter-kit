## Direction du parallax

Vous pouvez contrôler la direction du parallax avec `data-parallax-direction` :

Il faut préciser la hauteur du parent en CSS. 

```html
<div class="parallax-container">
  <img 
    src="image.jpg" 
    data-parallax 
    data-parallax-direction="y" 
    data-scrub="true"
  />
</div>
```

### Nécessite du CSS

```css
.parallax-container {
  position: relative;
  overflow: hidden;
}
```

## Vitesse du parallax

Vous pouvez contrôler la vitesse du parallax avec `data-speed` :

```html
<div class="parallax-container">
  <img 
    src="image.jpg" 
    data-parallax 
    data-parallax-direction="y" 
    data-scrub="true"
    data-speed="1.1"
  />
</div>
```

## Démarrage et fin de l'animation

Vous pouvez contrôler le démarrage et la fin de l'animation avec `data-start` et `data-end` :

```html
<div class="parallax-container">
    <img 
        src="https://picsum.photos/200/200?1" 
        data-parallax 
        data-parallax-direction="y" 
        data-speed="0.5" 
        data-scrub="true"
        data-start="top bottom+=20%"
        data-end="bottom top-=20%"
        class="parallax-element"
        alt="Parallax Image 1"
    >
</div>
```
